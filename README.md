Project Name: Spairo – Smart AI-app Powered Auto Parts 
Technologies: Flutter, Dart, Firebase, AI (Gemini Pro), Figma.
​Description: Developed a comprehensive mobile application to bridge the gap between car owners and spare parts suppliers.
​Key Contributions:
​Integrated AI-driven image recognition to identify spare parts via mobile camera.
​Implemented an intelligent Chatbot for automated customer assistance and part recommendations.
​Developed a dynamic search system using Part Name, VIN number, and compatibility analysis.
​Designed a multi-user architecture (Client App & Vendor Dashboard) for seamless order management and price comparison.
